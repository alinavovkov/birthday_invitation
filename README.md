Simple landing page scaffold

What you get
- `index.html` — minimal landing page
- `css/styles.css` — small responsive stylesheet
- `js/script.js` — tiny helper (smooth scroll, fake signup)

How to use

Open the `index.html` file in your browser, or serve the folder with a simple server.

To open in macOS from Terminal (zsh):

```bash
open index.html
```

To serve locally (Python 3):

```bash
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```
